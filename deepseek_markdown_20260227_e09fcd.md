# 1. Название проекта

NewsFlow
---

# 2. Краткое описание проекта

NewsFlow - это веб-агрегатор новостей, который собирает и отображает статьи из RSS-лент различных источников.
Приложение позволяет пользователям читать последние новости, искать материалы по ключевым словам, фильтровать их по категориям и сохравать понравившиеся статьи в избранное.
Система ориентирована на удобное и быстрое чтение новостей в браузере без необходимости посещать десятки разных сайтов.
---

# 3. Пользователь (кто и зачем использует)

Целевая аудитория:

- обычные читатели новостей
- люди, следящие за特定ными темами (IT, спорт, политика)
- исследователи и студенты, анализирующие медиа-поле

Цель использования:
- быстрое получение свежих новостей из разных источников
- чтение новостей на одной платформе
- поиск статей по интересующим темам
- сохранение интересных материалов для чтения позже

---

# 4. Функции (MVP)

В первую версию входят:

- Парсинг и отображение статей из RSS-лент (заранее заданные источники)
- Главная лента со всеми новостями (от новых к старым)
- Просмотр полного текста (или анонса) статьи на отдельной странице
- Фильтрация новостей по категориям / источникам
- Поиск статей по заголовку или ключевым словам
- Добавление статей в "Избранное" (сохранение в локальном хранилище)

---

# 5. Что не делаем (в первой версии)

- Личные кабинеты и авторизация пользователей
- Добавление пользовательских RSS-лент (кастомные источники)
- Комментарии к новостям
- Мобильное приложение (только адаптивная верстка для телефонов)
- Рекомендательные системы на основе ИИ

---

# 6. Платформа / стек (предварительно)

- Платформа: Web (браузерное приложение)
- Frontend:
    - Next.js (или React)
    - Tailwind CSS
    - shadcn/ui
- Backend (предварительно):
    - Next.js API Routes (Serverless функции) или Node.js
    - Библиотеки для парсинга RSS (например `rss-parser`)
    - База данных не требуется на старте (используем публичные API и локальное хранилище)
