# 1. Обязательные требования (MUST)

| № | Требование                                                  | Критерий приёмки                                                                                                                                        |
|---|-------------------------------------------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------|
| 1 | Загрузка и отображение ленты новостей из RSS                | При открытии главной страницы отображается список новостей из предустановленных источников; тест: лента не пуста, есть заголовки и даты                |
| 2 | Фильтрация по категориям/источникам                         | Пользователь может выбрать конкретную категорию; тест: при выборе категории отображаются только новости из неё                                         |
| 3 | Поиск статей по заголовку                                    | Пользователь может ввести текст в строку поиска; тест: список новостей фильтруется, показывая только статьи, содержащие введенный текст в заголовке   |
| 4 | Просмотр детальной страницы статьи                           | При клике на статью открывается страница с её полным текстом или описанием; тест: открывается отдельная страница с контентом и ссылкой на источник    |
| 5 | Добавление статьи в "Избранное"                              | На карточке статьи есть кнопка "В избранное"; тест: при нажатии статья сохраняется, иконка меняет состояние, статья появляется на странице "Избранное" |
| 6 | Просмотр списка "Избранное"                                  | Есть отдельная страница/раздел со всеми сохраненными статьями; тест: статьи, добавленные в избранное, отображаются в этом разделе                     |
| 7 | Авто-обновление контента (Pull-to-refresh или кнопка)        | Пользователь может инициировать принудительное обновление ленты; тест: при нажатии кнопки "Обновить" появляются новые статьи, если они есть           |

# 2. Желательные требования (NICE)

| № | Требование                                                        | Критерий приёмки                                                                                                 |
|---|-------------------------------------------------------------------|------------------------------------------------------------------------------------------------------------------|
| 1 | Бесконечная прокрутка (Infinite scroll) вместо пагинации         | При скролле вниз автоматически подгружаются следующие статьи; тест: страница не перезагружается, контент добавляется |
| 2 | Сохранение выбранных фильтров в URL                               | При фильтрации параметры (категория, поиск) отображаются в адресной строке; тест: можно поделиться ссылкой с применённым фильтром |
| 3 | Индикация "Прочитано" (визуальное выделение просмотренных статей) | Кликнутые ранее статьи отображаются с измененным стилем; тест: после перезагрузки страницы статус прочтения сохраняется |
| 4 | Поддержка темной/светлой темы интерфейса                          | Пользователь может переключать тему; тест: интерфейс меняется корректно                                          |
