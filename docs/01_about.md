# Этап 1. Описание проекта и границы

## 1. Название проекта
RSS News

## 2. Что делает приложение
RSS News — это простой веб-сайт, который собирает новости из нескольких RSS-источников и показывает их в одном интерфейсе. Пользователь может открыть общую ленту, выбрать конкретный источник, найти новость по заголовку и сохранить интересные статьи.

## 3. Пользователь
Пользователь — студент или обычный читатель, которому нужен простой способ просматривать новости из нескольких RSS-лент в одном месте без сложного интерфейса и лишних функций.

## 4. Функции
- загрузка новостей из RSS-источников;
- просмотр списка новостей;
- фильтрация по источнику;
- поиск по заголовку;
- просмотр страницы отдельной новости;
- сохранение статьи в список избранного.

## 5. Что не делаем
- личные аккаунты и авторизацию;
- комментарии пользователей;
- сложную систему рекомендаций;
- тяжёлые UI-библиотеки и анимации;
- полноценную CMS-панель.

## 6. Платформа / стек (предварительно)
Платформа: Web  
Репозиторий и совместная разработка: GitHub  
Язык: Python 3.11+  
Бэкенд-фреймворк: Flask  
Шаблоны: Jinja2  
Парсинг RSS: feedparser  
Хранение данных: SQLite  
Стили: простой CSS без тяжёлых UI-библиотек  
Запуск проекта: локально или на сервере с поддержкой Python/Flask
